⚙️  HOW IT WORKS
════════════════════════════════════════════════════════════════════════════════

1. SENDING MESSAGE (POST /api/send)
   ┌──────────────────────────────────────────────────────────────┐
   │ Input: { recipientDepartment, subject, message }             │
   └──────────────────────────────────────────────────────────────┘
                            ↓
   ┌──────────────────────────────────────────────────────────────┐
   │ Find existing Conversation or CREATE new one                 │
   │ Query: { faculty_id: userId, department: dept }             │
   └──────────────────────────────────────────────────────────────┘
                            ↓
   ┌──────────────────────────────────────────────────────────────┐
   │ CREATE new Message document:                                  │
   │ • conversation_id: linkto newly/existing conversation        │
   │ • sender_id: current faculty member                          │
   │ • message: trimmed content                                   │
   │ • type: "message"                                            │
   │ • is_read: false                                             │
   └──────────────────────────────────────────────────────────────┘
                            ↓
   ┌──────────────────────────────────────────────────────────────┐
   │ UPDATE Conversation metadata:                                 │
   │ • message_count++ (now 1, 2, 3, ...)                        │
   │ • last_message = messageId                                   │
   │ • last_message_at = now                                      │
   │ • unread_by_department++ (dep has new unread)               │
   └──────────────────────────────────────────────────────────────┘
                            ↓
   ┌──────────────────────────────────────────────────────────────┐
   │ Response: { success: true, data: { _id, ..., created_at } }  │
   └──────────────────────────────────────────────────────────────┘


2. RECEIVING MESSAGE (GET /api/my-messages)
   ┌──────────────────────────────────────────────────────────────┐
   │ Find all Conversations for this user                         │
   │ Query: { faculty_id: userId, status: { $ne: archived } }    │
   └──────────────────────────────────────────────────────────────┘
                            ↓
   ┌──────────────────────────────────────────────────────────────┐
   │ Find all Messages across those chapters                      │
   │ Query: { conversation_id: { $in: conversation_ids } }        │
   │ Sort: { created_at: -1 }                                     │
   │ Populate: sender details from User collection                │
   └──────────────────────────────────────────────────────────────┘
                            ↓
   ┌──────────────────────────────────────────────────────────────┐
   │ Enrich each message with conversation context                │
   │ Add: receiver_department, recipient_department, etc.         │
   └──────────────────────────────────────────────────────────────┘
                            ↓
   ┌──────────────────────────────────────────────────────────────┐
   │ Response: {                                                  │
   │   success: true,                                             │
   │   data: [ { _id, message, sender_name, created_at, ... } ], │
   │   conversations: [ { _id, department, message_count, ... } ] │
   │ }                                                             │
   └──────────────────────────────────────────────────────────────┘


3. REPLYING (POST /api/messages/reply/:messageId)
   ┌──────────────────────────────────────────────────────────────┐
   │ Input: { message: "..." }                                    │
   └──────────────────────────────────────────────────────────────┘
                            ↓
   ┌──────────────────────────────────────────────────────────────┐
   │ Find original Message by ID                                  │
   │ Populate its Conversation                                    │
   └──────────────────────────────────────────────────────────────┘
                            ↓
   ┌──────────────────────────────────────────────────────────────┐
   │ CREATE Reply Message:                                         │
   │ • conversation_id: (from original)                           │
   │ • reply_to: originalMessage._id  ← NEW!                   │
   │ • type: "reply"  ← NEW!                                    │
   │ • sender_id: department member's id                         │
   │ • message: reply content                                     │
   └──────────────────────────────────────────────────────────────┘
                            ↓
   ┌──────────────────────────────────────────────────────────────┐
   │ UPDATE Conversation:                                          │
   │ • message_count++ (now includes reply)                       │
   │ • last_message = replyId                                     │
   │ • unread_by_faculty++ (faculty has unread reply)            │
   └──────────────────────────────────────────────────────────────┘
                            ↓
   ┌──────────────────────────────────────────────────────────────┐
   │ Response: { success: true, data: replyMessage }              │
   └──────────────────────────────────────────────────────────────┘
